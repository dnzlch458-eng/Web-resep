Tentang Website Ini

Website Resep Nusantara dibuat sebagai tugas mata pelajaran TIK. Website ini berisi kumpulan resep masakan Indonesia dengan tampilan yang sederhana dan elegan.

Tujuan pembuatan website ini adalah untuk mempelajari dasar-dasar HTML dan CSS, serta memahami cara membuat website dengan beberapa halaman.

Fitur Website

  • Halaman beranda
  • Halaman daftar resep
  • Desain elegan dan rapi
  • Mudah dikembangkan

Pembuat

Website ini dibuat oleh siswa sebagai bagian dari tugas sekolah.
